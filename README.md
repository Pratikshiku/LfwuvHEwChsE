### 三、功能描述
系统分为（身份）：普通用户、护工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 收费标准
6. 接待信息
7. 房间信息
8. 入住信息
9. 请假信息
10. 提醒信息
11. 续住信息
12. 护理信息
13. 退住信息
14. 公告信息

**护工功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 收费标准
6. 接待信息
7. 房间信息
8. 入住信息
9. 请假信息
10. 提醒信息
11. 续住信息
12. 护理信息
13. 退住信息
14. 公告信息



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 护工管理
5. 收费标准管理
6. 接待信息管理
7. 房间类型管理

13. 护理信息管理
